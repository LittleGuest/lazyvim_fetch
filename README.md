# lazyvim_fetch

下载 https://www.lazyvim.org/ 相关插件到指定位置


```
LazyVim 安装更新脚本

Usage: lazyvim_fetch <COMMAND>

Commands:
  install  安装 windows 安装到 ~/AppData/Local/nvim ~/AppData/Local/nvim-data Linux 安装到 ~/.config/nvim/ ~/.local/share/nvim/lazy/
  update   更新starter和plugins
  delete   删除starter和plugins
  help     Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
